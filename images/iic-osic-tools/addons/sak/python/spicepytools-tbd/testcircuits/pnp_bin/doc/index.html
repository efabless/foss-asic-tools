PNP digital transitor test

Sometimes this type of transistor is called resistor equiped transistor (RET)

test schematic setups


 Test for the amplifier characteristics of the transistor.
  1. Input current over input voltage
  2. output current over input voltage.
This simulation is done at different temperatures and plotted in two linear plots.

postprocessing tests

The input current have to be lower than 1A. This test will catch transitors without any resistor at the input.
