bidirectional zener diode test

test schematic setup


 There are only two test simulations for a zener diode:
  1. the forward voltage
  2. the reverse voltage
Both simulations are done at different temperatures and plotted in two semilogy plots.

postprocessing tests

The expected range for the forward and the reverse voltage should be between 0 and 200 volts. It will catch all model errors where either the reverse voltage is not specified or the simulator takes a standard diode because of various model errors.
